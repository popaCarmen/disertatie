If input is an image and the output is an image, the process is DIGITAL IMAGE PROCESSING. 
If input is an image and the output is a description or information, the process is COMPUTER VISION.
If input is a description or code and the output is an image, the process is COMPUTER GRAPHICS.
If input is a description or code and the output is a description or information, the process is ARTIFICIAL INTELLIGENCE.
